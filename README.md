# Inventory Management System

A modern, full-stack inventory management application.

## Project Structure

```
InventoryManager/
├── frontend/          # React TypeScript application
│   └── README.md     # Frontend documentation
└── backend/          # (Future) Backend API
```

## Getting Started

### Frontend

Navigate to the frontend directory and follow the instructions in [frontend/README.md](./frontend/README.md)

```bash
cd frontend
npm install
npm run dev
```

## Features

- ✅ User authentication with protected routes
- ✅ Dashboard with inventory statistics
- ✅ Full CRUD operations for inventory
- ✅ Responsive design (mobile-first)
- ✅ Dark/Light theme support
- ✅ Modern UI with shadcn/ui
- ✅ State management with Redux Toolkit
- ✅ Data persistence
- ✅ Centralized API layer with React Query

## Tech Stack

### Frontend
- React 18 + TypeScript
- Vite
- shadcn/ui + Tailwind CSS
- Redux Toolkit + Redux Persist
- TanStack Query (React Query)
- React Router DOM
- Axios

### Backend (Coming Soon)
- Node.js + Express
- PostgreSQL / MongoDB
- JWT Authentication
- RESTful API
